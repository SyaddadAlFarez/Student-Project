The Mobile Banking Program is a program to facilitate banking services
where customers are required to log in first, using a username and 
password. After logging in the customer can take advantage of the 
features available in the mobile banking application such as checking 
balances, transfers between banks, Change pins, Purchases in the form 
of credit.
